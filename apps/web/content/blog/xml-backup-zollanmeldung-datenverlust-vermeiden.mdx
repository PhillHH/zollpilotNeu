# Die XML-Strategie: So verhindern Sie den totalen Datenverlust

Es passiert ohne Vorwarnung: Sie haben eine Stunde lang Daten in Atlas eingegeben, klicken auf den falschen Button – und alles ist weg. Die Session ist abgelaufen, der Browser abgestürzt, oder Atlas hat einfach beschlossen, dass es Zeit für eine Pause ist.

Wer schon einmal eine fast fertige Zollanmeldung verloren hat, weiß: Das darf nicht noch einmal passieren. Die Lösung heißt XML-Backup.

Atlas erlaubt es, Anmeldungen als XML-Dateien zu exportieren und später wieder zu importieren. Wer diese Funktion kennt und nutzt, spart sich im Ernstfall Stunden an Arbeit.
