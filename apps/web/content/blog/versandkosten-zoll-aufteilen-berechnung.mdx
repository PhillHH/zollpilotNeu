# Versandkosten-Puzzle: Wie man Liefergebühren zollkonform aufteilt

Eine Rechnung, fünf verschiedene Artikel, eine Pauschale für den Versand. Wie verteilt man die Versandkosten auf die einzelnen Positionen? Diese Frage stellt sich bei fast jeder Zollanmeldung mit mehreren Waren.

Der Zoll verlangt, dass Versandkosten zum Zollwert hinzugerechnet werden. Bei einer einzigen Ware ist das einfach. Bei mehreren Positionen wird es zum Rechenexempel.

Dieser Artikel zeigt, wie Sie Versandkosten korrekt aufteilen und welche Methoden der Zoll akzeptiert.
