# Ist das PDF eine offizielle Zollanmeldung?

Nein. Das von ZollPilot generierte PDF dient als **Ausfüllhilfe und Übersicht** für Ihre Unterlagen.

Es enthält alle relevanten Daten in strukturierter Form, ist aber **kein amtliches Formular**. Sie können das PDF nicht direkt beim Zoll einreichen. Stattdessen nutzen Sie die darin enthaltenen Informationen, um das offizielle Formular (online oder auf Papier) korrekt auszufüllen.
