# Web (Next.js)

App Router skeleton with:

- Landing page (`/`)
- Auth pages (`/login`, `/register`)
- Placeholders for `/app` and `/admin`
- Server-side health check call to the API
- Guarded `/app` and `/admin` via `/auth/me`
- Cases list/create in `/app/cases`

Run locally (without Docker):

1. `npm install`
2. `npm run dev`
